Documentation - Status

The Regno Standard was produced by Regno Software as an initial reference document to be developed and maintained by the Regno Open Data Standard Working Group (hereon known as the Working Group).

This draft documentation is intended to become the official Regno Standard. This documentation will remain a candidate document at least until 28th February 2018 in order to ensure the opportunity for wide review.

GitHub Issues are preferred for discussion of this specification. Please see the Support section for more details.

Documentation - Location

The latest version of this documentation will always be hosted online at the official Regno Standard website. The documentation will be publicly and freely available for download.

The master documentation documents are located in the public Regno Standard GitHub repository where all previous versions will be available.
