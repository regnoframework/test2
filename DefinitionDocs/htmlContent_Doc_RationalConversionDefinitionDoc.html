Definition Document - RationalConversionDefinitionDoc

The RationalConversionDefinitionDoc document is a definition document that contains the properties for a unique rational conversion instance.

This RationalConversionDefinitionDoc document is referenced by the ParamDefinitionDoc document and is used to convert parameter data values into converted display values.

The RationalConversionDefinitionDoc document does not contain a reference to the ConfigDoc document and therefore can be reused by multiple configurations and data sets.

Field Data Type Required Description
id String Yes A unique identifier that is referenced by all other documents that are associated with this definition. The proposed method of generation of this value is described in the section Unique Id Generation.
name String No A free form string value to name the conversion.
coefficients Array [64-bit Double] Yes An array of 64-bit Double coefficients that are used for the rational conversion operation.
identityDocIds Array [String] No An array of the unique identifiers (id) of each associated IdentityDoc document.
type Fixed String Yes The fixed value is always set to RationalConvDefDoc.
