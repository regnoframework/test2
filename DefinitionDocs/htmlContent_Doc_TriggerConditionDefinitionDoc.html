Definition Document - TriggerConditionDefinitionDoc

The TriggerConditionDefinitionDoc document is a definition document that contains the properties for a trigger condition instance.

The TriggerConditionDefinitionDoc document is referenced by the TriggerDefinitionDoc document.

The TriggerConditionDefinitionDoc document does not contain a reference to the ConfigDoc document and therefore can be reused by multiple configurations and data sets.

Field Data Type Required Description
id String Yes A unique identifier that is referenced by all other documents that are associated with this definition. The proposed method of generation of this value is described in the section Unique Id Generation.
name String No A free form string value to name the trigger condition.
index 32-bit Integer No The index of the condition. Can be used to place a condition in a specific index within a collection of trigger conditions definitions.
triggerType Fixed String No A fixed string representation of the trigger type. Valid options are: None, Power, Ignition, Data, External.
paramDefDocId String No The unique identifier (id) of the ParamDefinitionDoc document that holds the properties for the parameter associated with this trigger condition.
condition Fixed String No A fixed string representing the condition for the associated parameter. Valid options are: Equals, NotEquals, GreaterThan, LessThan, GreaterThanEqual, LessThanEqual.
threshold 64-bit Double No A 64-bit Double value representing the threshold value for the associated parameter and condition.
count 32-bit Integer No A 32-bit Integer value representing the number of counts that the threshold value and condition has to be valid for before the condition is recognised.
identityDocIds Array [String] No An array of the unique identifiers (id) of each associated IdentityDoc document.
type Fixed String Yes The fixed value is always set to TriggerCondDefDoc.
