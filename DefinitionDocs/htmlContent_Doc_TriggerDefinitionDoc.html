Definition Document - TriggerDefinitionDoc

The TriggerDefinitionDoc document is a definition document that contains the properties for a trigger instance.

The TriggerDefinitionDoc document is used to hold a set of trigger conditions that once met trigger the generation of data for a range of parameters at a set of fixed frequencies per parameter.

The TriggerDefinitionDoc document does not contain a reference to the ConfigDoc document and therefore can be reused by multiple configurations and data sets.

Field Data Type Required Description
id String Yes A unique identifier that is referenced by all other documents that are associated with this definition. The proposed method of generation of this value is described in the section Unique Id Generation.
name String No A free form string value to name the trigger.
startTriggerCondDefDocIds Array [String] Yes An array of identifiers (id) of the TriggerConditionDefinitionDoc documents that hold the properties for the start conditions associated with this trigger instance.
endTriggerCondDefDocIds Array [String] No An array of identifiers (id) of the TriggerConditionDefinitionDoc documents that hold the properties for the end conditions associated with this trigger instance.
startDelay 64-bit Long No A 64-bit Long value representing the start delay value in nanoseconds that is associated with this trigger instance.
stopDelay 64-bit Long No A 64-bit Long value representing the stop delay value in nanoseconds that is associated with this trigger instance.
rearms Boolean No A Boolean value representing if the trigger rearms after triggering. The default value is false (i.e. the trigger does not rearm).
output Fixed String Yes A fixed string representing the output stream of the trigger instance. Valid options are: Logged, Telemetry, TelemetryLogged.
paramDefDocIds Array [String] No An array of the unique identifiers (id) of each associated ParamDefinitionDoc document.
parameterFrequencies Array [64-bit Double] No An array of 64-bit Double values representing the frequency in hertz of each parameter represented in the paramDefIds array.
identityDocIds Array [String] No An array of the unique identifiers (id) of each associated IdentityDoc document.
type Fixed String Yes The fixed value is always set to TriggerDefDoc.
