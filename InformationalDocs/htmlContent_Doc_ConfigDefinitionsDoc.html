Informational Document - ConfigDefinitionsDoc

The ConfigDefinitionsDoc document is an informational document that defines the collection of definition documents that are used by the child documents in a given Configuration Document Collection.

Field Data Type Required Description
id String Yes A unique identifier that is referenced by the ConfigDoc document. The proposed method of generation of this value is described in the section Unique Id Generation.
configDocId String Yes The unique identifier (id) of the ConfigDoc document that this document is associated with.
paramDefDocIds Array [String] No An array of the unique identifiers (id) for each of the ParamDefinitionDoc documents that are used in the Configuration Document Collection.
textConvDefDocIds Array [String] No An array of the unique identifiers (id) for each of the TextConversionDefinitionDoc documents that are used in the Configuration Document Collection.
tableConvDefDocIds Array [String] No An array of the unique identifiers (id) for each of the TableConversionDefinitionDoc documents that are used in the Configuration Document Collection.
rationalConvDefDocIds Array [String] No An array of the unique identifiers (id) for each of the RationalConversionDefinitionDoc documents that are used in the Configuration Document Collection.
eventDefDocIds Array [String] No An array of the unique identifiers (id) for each of the EventDefinitionDoc documents that are used in the Configuration Document Collection.
canBusDefDocIds Array [String] No An array of the unique identifiers (id) for each of the CANBusDefinitionDoc documents that are used in the Configuration Document Collection.
triggerDefDocIds Array [String] No An array of the unique identifiers (id) for each of the TriggerDefinitionDoc documents that are used in the Configuration Document Collection.
type Fixed String Yes The fixed value is always set to ConfigDefsDoc.
