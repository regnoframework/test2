Data Document - ChannelSamplesDoc

The ChannelSamplesDoc document is a data document that contains the properties and values for a collection of data samples for a single data channel.

The ChannelSamplesDoc document is used to store raw sample values for user specified intervals. The document contains the field documentInterval that is used to specify the time range that the raw data samples values have sampled from.

Field Data Type Required Description
id String Yes A unique identifier that is generated from a hash of the document fields and values. The proposed method of generation of this value is described in the section Unique Id Generation.
configDocId String Yes The unique identifier (id) of the ConfigDoc document that this document is associated with.
paramDefDocId String Yes The unique identifier (id) of the ParamDefinitionDoc that holds the properties for the parameter associated with the data values stored in this document.
channelDefDocId String No The unique identifier (id) of the ChannelDefinitionDoc that holds the properties for the channel associated with the data values stored in this document.
startTime 64-bit Long Yes A 64-bit Long timestamp specifying the start time of the data that this document holds. The timestamp represents the number of nanoseconds since the reference time of midnight, 1st January 1970.
documentInterval 64-bit Long Yes A 64-bit Long value that specifies the interval in nanoseconds of the time range that the data stored in this document was sampled from.
sampleTimes Array [64-bit Long] Yes An array of 64-bit Long values. Each value represents the timestamp value of the sample at the associated array index. The timestamp represents the number of nanoseconds since the reference time of midnight, 1st January 1970.
sampleValues Array [64-bit Double] Yes An array of 64-bit Double values. Each value represents the sample value at the associated array index.
identityDocIds Array [String] No An array of the unique identifiers (id) of each associated IdentityDoc document.
type Fixed String Yes The fixed value is always set to ChannelSamplesDoc.
