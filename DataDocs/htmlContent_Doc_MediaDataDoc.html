Data Document - MediaDataDoc

The MediaDataDoc document is a data document that contains the properties and a source information for a media data file.

Field Data Type Required Description
id String Yes A unique identifier that is generated from a hash of the document fields and values. The proposed method of generation of this value is described in the section Unique Id Generation.
configDocId String Yes The unique identifier (id) of the ConfigDoc document that this document is associated with.
time 64-bit Long Yes A 64-bit Long timestamp specifying start time of the media file. The timestamp represents the number of nanoseconds since the reference time of midnight, 1st January 1970.
timeOffset 64-bit Long No A 64-bit Long value specifying the offset in nanoseconds that is applied to the time field value. The value can be both positive and negative.
source String No The source file name (including extension) for the media file.
sourceLocation String Yes The source folder path (excluding file name) for the media file.
sourceFileExtension String Yes An array of 64-bit Long values. Each value represents the timestamp value of the sample at the associated array index. The timestamp represents the number of nanoseconds since the reference time of midnight, 1st January 1970.
mediaType Fixed String Yes A fixed type description of the media. Valid options are: Audio, Video.
notes String No A free form string meta data field.
identityDocIds Array [String] No An array of the unique identifiers (id) of each associated IdentityDoc document.
type Fixed String Yes The fixed value is always set to MediaDataDoc.
