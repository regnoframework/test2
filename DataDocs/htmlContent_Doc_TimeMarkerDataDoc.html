Data Document - TimeMarkerDataDoc

The TimeMarkerDataDoc document is a data document that contains the properties and a data values for an individual time marker instance.

Field Data Type Required Description
id String Yes A unique identifier that is generated from a hash of the document fields and values. The proposed method of generation of this value is described in the section Unique Id Generation.
configDocId String Yes The unique identifier (id) of the ConfigDoc document that this document is associated with.
number 64-bit Long No An integer value associated with the marker instance.
time 64-bit Long Yes A 64-bit Long timestamp specifying time of the marker instance. The timestamp represents the number of nanoseconds since the reference time of midnight, 1st January 1970.
source String No A free form string describing the data source for the marker instance.
markerType String Yes A free form string describing the marker instance type.
identityDocIds Array [String] No An array of the unique identifiers (id) of each associated IdentityDoc document.
type Fixed String Yes The fixed value is always set to TimeMarkerDataDoc.
