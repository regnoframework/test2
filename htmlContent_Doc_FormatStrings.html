Documentation - Format Strings

For documents that contain a format field, a standardised string format is required to so that all Regno Standard documents maintain compatibility regardless of the language used to create the document.

For the Regno Standard, the format specifier that has been selected is the C format specifier.

By selecting the C format, the widest range of programming languages can be utilised for document processing interfaces as many languages already have built in support for C format specifiers though the use of printf and its derivatives.
